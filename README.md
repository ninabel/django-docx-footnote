# Django DOCX with footnotes Uploader 

Django admin plugin for uploading DOCX files, extracting formatted text with custom footnote processing, and storing as HTML in a TextField.

Features
* *Admin integration* - Upload DOCX directly in Django admin

* *Custom footnote formatting* - Transforms standard footnotes to styled HTML using django-like template

* *Removes excess style attributes* - Clean output without default styling

* *Model field storage* - Stores formatted HTML in TextField
